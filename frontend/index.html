❓ Help
What is WiFi?

WiFi (802.11) is a wireless networking standard that uses radio waves to provide high-speed internet and network connections.

What do the bands mean?

2.4 GHz has longer range but slower speed. 5 GHz has shorter range but faster speed. Both are used by modern routers.

What is a channel?

WiFi channels are sub-divisions of the frequency band. Using a less crowded channel improves performance.

Is data logged?

Yes — the Python backend logs all scan events to /logs/ as CSV and JSON Lines automatically.

How to start backend?

cd backend && pip install -r requirements.txt && python main.py

📶 WiFi Protocol

802.11 a/b/g/n/ac/ax. Uses 2.4 GHz and 5 GHz bands. Channels 1-14 (2.4G) and 36-165 (5G).

🔒 Security

WEP (broken), WPA (legacy), WPA2 (standard), WPA3 (latest). Always use WPA2 or WPA3.

🐍 Backend

FastAPI + scapy/pywifi. WS: ws://localhost:8002/ws. REST: /api/. Logs: /logs/.

📜 Log

📶 Lesson 1: What is WiFi?

WiFi uses radio waves to connect devices to the internet wirelessly. Your router broadcasts a signal that your phone, laptop, and smart devices can connect to.

WiFi uses two main frequency bands:

2.4 GHz — longer range, slower speed, more interference 5 GHz — shorter range, faster speed, less interference 📱 Phone ──radio waves──▶ 📡 Router ──cable──▶ 🌐 Internet

🔒 Lesson 2: WiFi Security

WiFi security has evolved over the years:

WEP — 1997 — BROKEN (can be cracked in minutes) WPA — 2003 — Legacy (TKIP, vulnerable) WPA2 — 2004 — Standard (AES encryption) WPA3 — 2018 — Latest (stronger protection) Open = No password = Anyone can see your data!

Always use WPA2 or WPA3 encryption. Never connect to open networks for sensitive tasks.

📻 Lesson 3: WiFi Channels

WiFi channels are like lanes on a highway. If too many routers use the same channel, they interfere with each other and slow down.

📏 Lesson 4: RSSI — Signal Strength

RSSI (Received Signal Strength Indicator) tells you how strong a WiFi signal is. Measured in dBm:

-30 dBm = EXCELLENT (next to router) -50 dBm = GOOD (same room) -70 dBm = OK (through walls) -80 dBm = WEAK (far away) -90 dBm = LOST (barely usable)

Walls, floors, and other objects weaken the signal. Metal and concrete block WiFi the most.
